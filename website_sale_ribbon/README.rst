===================
Website Sale Ribbon
===================

This module add the functionality of defined ecommerce ribbons to use in products as shop. We can define both ribbon text and ribbon color.


Installation
============

To install this module, you need to:

#. Only need to install the module

Configuration
=============

To configure this module, you need to:

#. Nothing to configure

Usage
=====

To use this module, you need to:

#. Go to Website / Catalog / Ribbons and create new ribbons
#. In order to use go to Products form view, and in the Sales tab  set Ribbon field with the ribbon you want to use. not that Styles field should had been set "Show Ribbon" style in order to the ribbon can be show.
